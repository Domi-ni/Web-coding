HTML

o que é

  • não é uma linguagem de programação
  • é uma linguagem de marcação
  • é uma forma de organizar o conteudo (img, texto e etc) em uma pagina web

sintaxe

  • As marcações sao chamadas de Tags
  • são escritos da segunte forma: "<*nomedatag>" <*/nomedatag>"

Tags

  • Existem diversas tags dentro do HTML
  • dentre elas temos:

tags de titulo

  • Para indicar que um texto é um título, utilizamos as tags de heading
  • contém 6 níveis <*h1> a <*h6>
  • Seguem uma ordem de importância em ordem decrescente
  • h1 é maior e h6 menor
  • <*h1> nome, título principal da página
  • <*h2>  subtítulo ou título de seções dentro de um documento
  • OBS <*h1> SO PODE SER UTILIZADO UMA VEZ POR PAGINA

tags de paragrafo

    Utilizada para textos comuns Tag <*p> <*/p>

tags de marcação de enfase

    A tag <*strong>* e <*b> deixam o texto em negrito <*em> e <*i> sao utilizadas para italico A utilização das tags <*strong>* e <*em> sao mais indicadas pois dizem exatamente a intenção

Imagens

    a tag <*img> indica onde a imagem sera renderizada Dois atributos sao gerados src: aponta para a localização da imagem (no computador ou na web) atributo obrigatorio alt: descrição da imagem para leitores de tela e largura (width) e altura(height), especificam a largura e a altura da imagem em pixels • <*figcaption>: adiciona uma legenda para a imagem
